## How to use

### Install

```bash
npm i -D vuepress-plugin-add-this
```

Or

```bash
yarn add -D vuepress-plugin-add-this
```

### Usage

```js {3-7}
// .vuepress/config.js
module.exports = {
  plugin: [
    "add-this",
    {
      pubid: "your pubid",
    },
  ],
};
```
