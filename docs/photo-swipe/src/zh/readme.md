让您的图像支持预览，缩放，共享，滑动查看和下载。

## 使用插件

### 安装

```bash
npm i -D vuepress-plugin-photo-swipe
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugin: ["photo-swipe"],
};
```
