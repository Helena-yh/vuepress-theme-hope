## [Offical Docs](https://vuepress-add-this.mrhope.site/) / [官方文档](https://vuepress-add-this.mrhope.site/zh/)

## How to use

### Install

```bash
npm i -D vuepress-plugin-add-this
```

Or

```bash
yarn add -D vuepress-plugin-add-this
```

### Usage

```js
// .vuepress/config.js
module.exports = {
  plugin: [
    "add-this",
    {
      pubid: "your pubid",
    },
  ],
};
```
